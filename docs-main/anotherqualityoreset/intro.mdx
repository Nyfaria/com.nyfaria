# Another Quality Ore Set

**Another Quality Ore Set** adds three new ore tiers to Minecraft: Easium, Medium, and Hardium, plus a powerful end-game material called Telos. Each tier provides a complete set of tools, armor, and special items like hammers, paxels, and tree axes.

## Features

- **Easium Tier** - Entry-level custom ore with full tool and armor set
- **Medium Tier** - Mid-tier ore with balanced stats
- **Hardium Tier** - High-tier ore for advanced players
- **Telos Tier** - End-game material crafted from the other tiers, featuring powerful armor and weapons

### Special Tools

- **Hammer** - A powerful mining tool that breaks blocks in a 3x3 area
- **Paxel** - A combination of pickaxe, axe, and shovel in one tool
- **Tree Axe** - Chops down entire trees at once
- **Rod** - A crafting material for each tier
