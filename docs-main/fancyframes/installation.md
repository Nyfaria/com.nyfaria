This page will document how to install FancyFrames.

Quick install (placeholder):

1. Download the mod jar from the [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fancy-frames) or [Modrinth](https://modrinth.com/project/fancy-frames).
2. Place the jar into your `.minecraft/mods/` folder (Fabric or NeoForge as appropriate).
3. Start Minecraft with the correct Fabric/NeoForge loader.
